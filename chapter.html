DHT11 Sensor 📡

📘 Introduction

DHT11 ek temperature aur humidity sensor hai. Ye Arduino ke saath use hota hai.

⚡ Working

Ye sensor environment ka temperature detect karta hai aur Arduino ko digital signal bhejta hai.

🔌 Connections

VCC → 5V
GND → GND
DATA → Pin 2

💻 Arduino Code

void setup(){ Serial.begin(9600); } void loop(){ Serial.println("DHT11 Working"); delay(1000); }

🚀 Applications

• Weather Station
• Smart Home
• IoT Projects
• Temperature Monitoring
