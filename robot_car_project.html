⚡ Arduino Robot Car Project

🤖 Robot Car

Arduino se 4-wheel robot car banao — L298N motor driver ke saath

PROJECT #05 · INTERMEDIATE

1. 🤖 Project Introduction

Robot Car project mein Arduino Uno ek L298N motor driver ke zariye 2 DC motors ko control karta hai — jisse car aage, peeche, left aur right move karti hai. Ek ultrasonic sensor (HC-SR04) obstacle detect karta hai — agar koi object saamne aaye toh car automatically ruk jaati hai ya direction change kar leti hai. Remote control ke liye Bluetooth (HC-05) bhi add kar sakte ho!

💡 Fun Fact: L298N ek H-Bridge circuit hai jo motor ko dono directions mein chalata hai — aur PWM se speed bhi control hoti hai (0–255 range)!

2. 📋 Components Required

  • 🟩 Arduino Uno × 1
  • ⚙️ L298N Dual Motor Driver Module × 1
  • 🔵 DC Motors (with wheels) × 2 (ya 4 motor chassis kit)
  • 📡 HC-SR04 Ultrasonic Sensor × 1 (obstacle detection)
  • 🔋 Li-ion / 9V Battery Pack × 1
  • 🔗 Jumper Wires × 10+
  • 🧱 Robot Car Chassis Kit × 1
  • 🔌 USB Cable × 1
⚡ L298N Pins:
ENA → PWM Pin 5  |  IN1 → Pin 6  |  IN2 → Pin 7
ENB → PWM Pin 10  |  IN3 → Pin 8  |  IN4 → Pin 9

📡 HC-SR04: VCC → 5V | GND → GND | TRIG → Pin 2 | ECHO → Pin 3

3. 🔌 Circuit Diagram

📌 ENA/ENB → PWM pins (speed)  |  IN1–IN4 → direction  |  HC-SR04 → Pin 2 (TRIG), Pin 3 (ECHO)

* Battery 7–12V use karo L298N ke VCC mein; Arduino ko USB ya L298N ke 5V output se power do

4. 💻 Arduino Code

📦 Library: Koi extra library zarurat nahi — sirf standard Arduino functions use honge! 
// ============================================
// Robot Car - Arduino Uno
// L298N Motor Driver + HC-SR04 Obstacle Avoid
// By: Arduino Projects
// ============================================

// ---- Motor A (Left) ----
#define ENA   5   // PWM speed control
#define IN1   6   // Direction 1
#define IN2   7   // Direction 2

// ---- Motor B (Right) ----
#define ENB   10  // PWM speed control
#define IN3   8   // Direction 1
#define IN4   9   // Direction 2

// ---- Ultrasonic Sensor ----
#define TRIG_PIN  2
#define ECHO_PIN  3

// ---- Settings ----
#define SPEED         180   // 0–255 (speed)
#define TURN_SPEED    150   // Turning speed
#define OBSTACLE_DIST 20    // 20 cm se kam = ruko!

long duration;
int  distance;

// ============================================
void setup() {
  // Motor pins output
  pinMode(ENA, OUTPUT); pinMode(IN1, OUTPUT); pinMode(IN2, OUTPUT);
  pinMode(ENB, OUTPUT); pinMode(IN3, OUTPUT); pinMode(IN4, OUTPUT);

  // Ultrasonic pins
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);

  Serial.begin(9600);
  Serial.println("=== Robot Car Ready! ===");
  delay(1000);
}

// ============================================
void loop() {
  distance = getDistance();

  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");

  if (distance > 0 && distance <= OBSTACLE_DIST) {
    // Obstacle detected!
    stopCar();
    Serial.println("⚠️ Obstacle! Stopping...");
    delay(500);

    // Peeche jao thoda
    moveBackward();
    delay(400);
    stopCar();
    delay(200);

    // Right turn karo
    turnRight();
    delay(500);
    stopCar();
    delay(200);

  } else {
    // Raasta clear — aage jao
    moveForward();
    Serial.println("✅ Moving Forward");
  }

  delay(100);
}

// ============================================
// MOVEMENT FUNCTIONS
// ============================================

void moveForward() {
  analogWrite(ENA, SPEED);
  analogWrite(ENB, SPEED);
  digitalWrite(IN1, HIGH); digitalWrite(IN2, LOW);   // Motor A forward
  digitalWrite(IN3, HIGH); digitalWrite(IN4, LOW);   // Motor B forward
}

void moveBackward() {
  analogWrite(ENA, SPEED);
  analogWrite(ENB, SPEED);
  digitalWrite(IN1, LOW);  digitalWrite(IN2, HIGH);  // Motor A backward
  digitalWrite(IN3, LOW);  digitalWrite(IN4, HIGH);  // Motor B backward
}

void turnLeft() {
  analogWrite(ENA, TURN_SPEED);
  analogWrite(ENB, TURN_SPEED);
  digitalWrite(IN1, LOW);  digitalWrite(IN2, HIGH);  // Motor A backward
  digitalWrite(IN3, HIGH); digitalWrite(IN4, LOW);   // Motor B forward
}

void turnRight() {
  analogWrite(ENA, TURN_SPEED);
  analogWrite(ENB, TURN_SPEED);
  digitalWrite(IN1, HIGH); digitalWrite(IN2, LOW);   // Motor A forward
  digitalWrite(IN3, LOW);  digitalWrite(IN4, HIGH);  // Motor B backward
}

void stopCar() {
  analogWrite(ENA, 0);
  analogWrite(ENB, 0);
  digitalWrite(IN1, LOW); digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW); digitalWrite(IN4, LOW);
}

// ============================================
// ULTRASONIC DISTANCE
// ============================================

int getDistance() {
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);

  duration = pulseIn(ECHO_PIN, HIGH, 30000); // 30ms timeout
  if (duration == 0) return 999; // No echo = clear
  return duration * 0.034 / 2;  // cm mein convert
}

6. ✅ Testing

Car ko kisi flat surface pe rakho. Power on karo — car aage chalani chahiye. Saamne haath rakho — 20cm ke andar aao toh car ruk jaani chahiye.

🎉 Success: Car move ho rahi hai aur obstacles se bach rahi hai? Congratulations — project complete!
🔧 Motor ulti chal rahi hai? IN1/IN2 ya motor ke wires swap karo.
⚡ Slow hai? SPEED value badha do (max 255).

7. 🛠️ Common Errors & Fix

  • Car nahi chal rahi? → ENA/ENB jumper hata do — ya PWM se HIGH karo
  • Ek motor nahi chal rahi? → L298N ke IN1/IN2 ya IN3/IN4 check karo
  • Ultrasonic kaam nahi kar raha? → TRIG/ECHO wires swap karke dekho
  • Car circle mein ghoom rahi? → Dono motors ki speed match karo (ENA/ENB PWM value)
  • Car bahut slow hai? → Battery fully charged hai? SPEED value 180+ rakho
  • Arduino reset hota hai? → Motors ke liye alag power supply use karo
