Push Button 🚀

📘 Introduction

Push Button ek input component hai jo press karne par signal bhejta hai.

📍 Pins Information

Push Button ke 4 pins hote hain aur normally open connection use hota hai.

⚡ Voltage & Current

Normally 3.3V aur 5V circuits me use hota hai.

🖼️ Component Diagram

Push Button wiring aur breadboard connection diagram yahan show hoga.

🌍 Applications

Input systems, menu control, gaming aur Arduino projects me use hota hai.

🚀 Advantages

Simple, low cost aur easy to use input device.
