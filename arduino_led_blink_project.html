1. 💡 Project Introduction

LED Blink Arduino ka sabse basic aur beginner-friendly project hai. Is project mein Arduino Uno Pin 13 par connected LED ko 1 second ON aur 1 second OFF karta hai — baar baar. Yeh project coding, wiring, aur upload — teeno ka practice karwata hai.

💡 Fun Fact: Arduino IDE mein pehle se "Blink" example available hai: File → Examples → 01.Basics → Blink

2. 📋 Components Required

  • 🟩 Arduino Uno × 1
  • 💡 LED (Red ya Green) × 1
  • ⚡ 220Ω Resistor × 1
  • 🔗 Jumper Wires × 2
  • 🧱 Breadboard × 1
  • 🔌 USB Cable × 1
⚡ Resistor kyun? Arduino Pin 13 max 40mA deta hai, LED ko sirf ~20mA chahiye. 220Ω resistor current limit karta hai: I = 5V ÷ 220Ω ≈ 22mA ✓

4. 💻 Arduino Code

// LED Blink Project - Arduino Uno
// Pin 13 pe LED connect karo
 
const int ledPin = 13;  // LED pin
int delayTime = 1000;   // 1 second
 
void setup() {
  pinMode(ledPin, OUTPUT);   // Pin 13 OUTPUT set karo
  Serial.begin(9600);
  Serial.println("LED Blink Start!");
}
 
void loop() {
  digitalWrite(ledPin, HIGH); // LED ON
  Serial.println("LED: ON");
  delay(delayTime);
 
  digitalWrite(ledPin, LOW);  // LED OFF
  Serial.println("LED: OFF");
  delay(delayTime);
}

7. 🛠️ Common Errors & Fix

  • LED nahi jali? → LED polarity check karo (lamba leg = anode = +)
  • Upload fail? → USB cable aur COM port check karo
  • "Board not found"? → Tools → Board → Arduino Uno select karo
  • LED dim? → Resistor sahi lagao (220Ω)
💡 Shortcut: Arduino Uno pe pehle se Pin 13 onboard LED hai — bina wiring ke bhi test kar sakte ho!
