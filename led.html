LED Blink Project 💡

📘 Introduction

LED Blink Arduino ka beginner project hai. Is project me LED ON aur OFF hoti hai. Ye project Arduino coding aur electronics seekhne ke liye best hai.

🔌 Circuit Diagram

💻 Arduino Code


int led = 13;

void setup(){

 pinMode(led, OUTPUT);

}

void loop(){

 digitalWrite(led, HIGH);

 delay(1000);

 digitalWrite(led, LOW);

 delay(1000);

}

🧰 Parts List

🟩 Arduino UNO
💡 LED
⚡ 220Ω Resistor
🔗 Jumper Wires
🧱 Breadboard
🔌 USB Cable

⚙️ Wiring Steps

1️⃣ LED ko breadboard me lagao.
2️⃣ LED anode ko resistor se connect karo.
3️⃣ Resistor ko Arduino Pin 13 se connect karo.
4️⃣ LED cathode ko GND se connect karo.
5️⃣ Arduino ko USB cable se connect karo.
6️⃣ Code upload karo aur LED blink dekho.
