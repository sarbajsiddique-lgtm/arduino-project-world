1. 📡 Project Introduction

Ultrasonic Sensor (HC-SR04) Arduino ka ek bahut popular project hai. Yeh sensor sound waves (ultrasonic) bhejta hai aur unka echo wapas receive karta hai — is time difference se distance calculate hoti hai. Is project mein Arduino Uno HC-SR04 se distance measure karke Serial Monitor par cm mein display karega.

💡 Fun Fact: HC-SR04 ka range 2cm se 400cm tak hota hai, accuracy ±3mm hoti hai — bilkul bat ki tarah kaam karta hai! 🦇

4. 💻 Arduino Code

// Ultrasonic Sensor Project - Arduino Uno
// HC-SR04 — Distance Measurement
// TRIG → Pin 9 | ECHO → Pin 10
 
const int trigPin = 9;   // TRIG pin
const int echoPin = 10;  // ECHO pin
 
long duration;           // Sound wave travel time
float distanceCm;        // Distance in cm
float distanceInch;      // Distance in inches
 
void setup() {
  pinMode(trigPin, OUTPUT);  // TRIG = OUTPUT
  pinMode(echoPin, INPUT);   // ECHO  = INPUT
  Serial.begin(9600);
  Serial.println("=== Ultrasonic Sensor Ready ===");
}
 
void loop() {
  // TRIG pin ko trigger karo
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);   // 10 microsecond pulse
  digitalWrite(trigPin, LOW);
 
  // ECHO pin se time measure karo
  duration = pulseIn(echoPin, HIGH);
 
  // Distance calculate karo
  // Sound speed = 343 m/s = 0.0343 cm/us
  // Distance = (duration / 2) * 0.0343
  distanceCm    = (duration / 2.0) * 0.0343;
  distanceInch  = distanceCm / 2.54;
 
  // Serial Monitor pe print karo
  Serial.print("Distance: ");
  Serial.print(distanceCm, 1);
  Serial.print(" cm  |  ");
  Serial.print(distanceInch, 1);
  Serial.println(" inches");
 
  // Alert: object 10cm se paas ho
  if (distanceCm < 10) {
    Serial.println("⚠️  WARNING: Object Too Close!");
  }
 
  delay(500);  // 0.5 sec baad phir measure karo
}

7. 🛠️ Common Errors & Fix

  • Distance 0 aa rahi? → TRIG aur ECHO pins check karo (9 aur 10 hone chahiye)
  • Galat value? → Wires dhang se connect hain? VCC = 5V (3.3V nahi)
  • "Upload fail"? → USB cable aur COM port check karo
  • Values random hain? → Sensor ke saamne clear space rakho, koi obstacle nahi
  • Range kam lag rahi? → HC-SR04 soft/fabric objects detect nahi karta achhe se
💡 Shortcut: pulseIn() timeout set karo — pulseIn(echoPin, HIGH, 30000) — 30ms se zyada wait nahi karega (400cm+ range ignore)
