Potentiometer 🚀

📘 Introduction

Potentiometer ek variable resistor hai jo resistance manually change karta hai.

📍 Pins Information

Potentiometer ke 3 pins hote hain — VCC, GND aur Signal pin.

⚡ Voltage & Current

Normally 3.3V aur 5V Arduino circuits me use hota hai.

🖼️ Component Diagram

Potentiometer wiring aur analog connection diagram yahan show hoga.

🌍 Applications

Volume control, brightness control aur analog input projects me use hota hai.

🚀 Advantages

Easy control, adjustable resistance aur beginner friendly.
