1. 🌡️ Project Introduction

Temperature Sensor project mein Arduino Uno DHT11 sensor se temperature aur humidity dono measure karta hai. DHT11 ek digital sensor hai jo accurate aur reliable readings deta hai. Is project mein temperature Celsius aur Fahrenheit dono mein Serial Monitor par display hogi — aur agar temperature zyada ho toh LED alert bhi milega!

💡 Fun Fact: DHT11 temperature 0°C – 50°C range mein aur humidity 20% – 80% range mein measure kar sakta hai — ±2°C accuracy ke saath!

2. 📋 Components Required

  • 🟩 Arduino Uno × 1
  • 🌡️ DHT11 Temperature & Humidity Sensor × 1
  • ⚡ 10kΩ Pull-up Resistor × 1
  • 💡 Red LED × 1 (heat alert ke liye)
  • ⚡ 220Ω Resistor × 1 (LED ke liye)
  • 🔗 Jumper Wires × 5
  • 🧱 Breadboard × 1
  • 🔌 USB Cable × 1
⚡ DHT11 ke 3 pins (module version):
VCC → 5V  |  GND → GND  |  DATA → Pin 2

⚡ DHT11 ke 4 pins (bare sensor version):
Pin 1 → 5V  |  Pin 2 → Data (Pin 2 + 10kΩ to VCC)  |  Pin 3 → NC  |  Pin 4 → GND

4. 💻 Arduino Code

📦 Library Install karo pehle: Arduino IDE → Tools → Manage Libraries → "DHT sensor library" by Adafruit install karo 
// Temperature & Humidity Monitor - Arduino Uno
// DHT11 Sensor | DATA → Pin 2 | LED Alert → Pin 8
// Library: DHT sensor library by Adafruit
 
#include <DHT.h>
 
#define DHTPIN     2      // DHT11 DATA pin
#define DHTTYPE    DHT11  // Sensor type
#define LED_PIN    8      // Heat alert LED
#define TEMP_LIMIT 35.0   // 35°C se zyada = alert!
 
DHT dht(DHTPIN, DHTTYPE);
 
float tempC    = 0.0;  // Temperature Celsius
float tempF    = 0.0;  // Temperature Fahrenheit
float humidity = 0.0;  // Humidity %
 
void setup() {
  pinMode(LED_PIN, OUTPUT);
  Serial.begin(9600);
  dht.begin();
  Serial.println("=== Temperature Monitor Ready ===");
  delay(2000);  // DHT11 startup time
}
 
void loop() {
  // Sensor se data padho
  humidity = dht.readHumidity();
  tempC    = dht.readTemperature();       // Celsius
  tempF    = dht.readTemperature(true);   // Fahrenheit
 
  // Reading valid hai ya nahi check karo
  if (isnan(humidity) || isnan(tempC)) {
    Serial.println("❌ Sensor Error! Wiring check karo.");
    delay(2000);
    return;
  }
 
  // Serial Monitor pe print karo
  Serial.println("================================");
  Serial.print("🌡️  Temperature : ");
  Serial.print(tempC, 1);
  Serial.print(" °C  |  ");
  Serial.print(tempF, 1);
  Serial.println(" °F");
 
  Serial.print("💧 Humidity    : ");
  Serial.print(humidity, 1);
  Serial.println(" %");
 
  // Heat Index calculate karo
  float heatIndex = dht.computeHeatIndex(tempF, humidity);
  Serial.print("🔥 Heat Index  : ");
  Serial.print(heatIndex, 1);
  Serial.println(" °F");
 
  // Temperature alert
  if (tempC >= TEMP_LIMIT) {
    digitalWrite(LED_PIN, HIGH);  // LED ON
    Serial.println("⚠️  WARNING: High Temperature Alert!");
  } else {
    digitalWrite(LED_PIN, LOW);   // LED OFF
    Serial.println("✅ Temperature Normal");
  }
 
  Serial.println("================================");
  delay(2000);  // Har 2 second mein read karo
}

7. 🛠️ Common Errors & Fix

  • "Sensor Error" aa raha? → DATA pin aur wiring check karo, 10kΩ pull-up resistor lagao
  • NaN values? → DHT library sahi install hai? Adafruit DHT library use karo
  • Galat temperature? → DHT11 ko 2 second startup time do — pehli reading skip karo
  • LED nahi jali? → Pin 8 check karo + 220Ω resistor lagao
  • Compile error? → #include <DHT.h> check karo — library installed honi chahiye
💡 Upgrade: DHT11 ki jagah DHT22 use karo — zyada accurate (±0.5°C), wide range (-40°C to 80°C), aur code same rahega! Sirf #define DHTTYPE DHT22 change karo.
